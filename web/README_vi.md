Đây là dự án [Next.js](https://nextjs.org) được khởi tạo (bootstrapped) bằng [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Bắt đầu

Đầu tiên, chạy development server:

```bash
npm run dev
# hoặc
yarn dev
# hoặc
pnpm dev
# hoặc
bun dev
```

Mở [http://localhost:3000](http://localhost:3000) bằng browser của bạn để xem kết quả.

Bạn có thể bắt đầu chỉnh sửa trang bằng cách sửa đổi `app/page.tsx`. Trang này tự động cập nhật khi bạn chỉnh sửa file.

Dự án này sử dụng [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) để tự động tối ưu hóa và tải [Geist](https://vercel.com/font), một bộ font mới cho Vercel.

## Tìm hiểu thêm

Để tìm hiểu thêm về Next.js, hãy xem các tài nguyên sau:

- [Tài liệu Next.js](https://nextjs.org/docs) - tìm hiểu về các tính năng của Next.js và API.
- [Học Next.js](https://nextjs.org/learn) - một bài hướng dẫn (tutorial) Next.js tương tác.

Bạn có thể xem [GitHub repository của Next.js](https://github.com/vercel/next.js) - chúng tôi luôn hoan nghênh phản hồi (feedback) và đóng góp của bạn!

## Deploy trên Vercel

Cách dễ nhất để deploy ứng dụng Next.js của bạn là sử dụng [Vercel Platform](https://vercel.com/new?utm_medium=default-template&filter=next.js&utm_source=create-next-app&utm_campaign=create-next-app-readme) từ những người tạo ra Next.js.

Xem [tài liệu deployment Next.js](https://nextjs.org/docs/app/building-your-application/deploying) của chúng tôi để biết thêm chi tiết.
